# AI Resume Analyzer 🤖

An AI-powered full stack web app that analyzes your resume against a job description and gives you a match score, skill gaps, keyword analysis, and an AI-rewritten summary.

## Live Demo
👉 https://my-resume-ai-omega.vercel.app

## Features
- 📄 Upload a PDF or paste resume text
- 🧠 AI-powered analysis using LLaMA 3.3 via Groq API
- 📊 Match score out of 100
- ✅ Strengths and ⚠️ gaps breakdown
- 🔑 Matched and missing keywords
- ✍️ AI-rewritten professional summary
- 📋 Copy to clipboard

## Tech Stack
- **Frontend** — Next.js, Tailwind CSS
- **AI** — Groq API, LLaMA 3.3-70B
- **PDF Parsing** — pdf2json
- **Deployment** — Vercel
